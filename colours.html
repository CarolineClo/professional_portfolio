Assignment Details

The purpose of this assignment is to demonstrate an understanding of a harmonious colour pallet and how to use an animation library to create interesting user interactions on a mock up landing page. I chose colour pallet and applied contrasting colours to background and text for easy readability, I also chose an accent colour so interactive elements stand out in the page. I then created a dark mode function for accessibility and for people who prefer a less bright screen. I used Web Animations API to create a hover animation on the CTA in the header and I experimented Motion One to create a scrolling animation on the navbar.
