Outline

For this group project we worked together with an independent ceramic studio on Bornholm, creating an online prototype for a new website, shop together a social media strategy. Our final outcome is based on research of Gerner Jahnckeâ€™s target market and current design and social media trends. The website fetches data from a data base we created using Wordpress of products categorised by collections and materials

Details

Mobile View

Each page is fully responsive for mobile and tablet view. The landing page features an image carousel on mobile view giving access to many images without endless scrolling.

Texture in Images

We incorporated close up shots throughout the website to show the texture detail of the pieces. This creates the illusion of a more tactile experience, which our research said many online shoppers were missing.

Dynamic data

The javascript is written to fetch product details from a database created on Wordpress. The product page then filters the view, based on the users choice of collection or product.
